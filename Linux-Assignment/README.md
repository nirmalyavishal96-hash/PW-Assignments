## Topics Covered

The assignment demonstrates practical understanding of the following Linux concepts:

* Introduction to Linux
* Hard Links vs Soft Links
* Linux Kernel Overview
* User Account Creation
* `grep` Command Usage
* User and Group Management
* Directory Ownership and Permissions
* Identifying and Terminating High CPU Processes

---

## Practical Tasks Implemented

The lab includes hands-on tasks such as:

* Creating a user (`p1`)
* Creating multiple groups (`g1`, `g2`, `g3`)
* Assigning a primary group to a user
* Managing file and directory permissions
* Setting directory ownership
* Monitoring system processes
* Terminating resource-heavy processes
