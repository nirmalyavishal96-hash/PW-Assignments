# Python Functions Assignment

## Overview

This repository contains solutions for the **Python Functions Assignment**.
The assignment focuses on implementing core Python programming concepts such as:

* Functions
* Iterators
* Generators
* Lambda functions
* `map()` and `filter()` functions

The solutions are implemented in a **Google Colab / Jupyter Notebook** format where each question is explained with its corresponding Python code.

## Practical Questions Implemented

The notebook includes solutions for the following programming tasks:

1. Python function to calculate the **sum of even numbers in a list**
2. Function to **reverse a string**
3. Function to **return squares of numbers in a list**
4. Program to **check prime numbers between 1 and 200**
5. Custom **iterator class for Fibonacci sequence**
6. Generator function to **yield powers of 2**
7. Generator function to **read a file line by line**
8. Sorting tuples using a **lambda function**
9. Temperature conversion from **Celsius to Fahrenheit using `map()`**
10. Removing vowels from a string using **`filter()`**
11. Accounting routine using **`lambda` and `map()`**
