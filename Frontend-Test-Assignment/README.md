# Frontend Load Testing using Apache JMeter

## Overview

This repository demonstrates **load testing of a web application using Apache JMeter**.
The goal of this project is to simulate multiple users accessing a web application and analyze the performance metrics to identify potential bottlenecks.

Load testing helps evaluate system stability, response time, and throughput under high traffic conditions.

---


## Tool Used

Apache JMeter is an open-source tool used for performance and load testing of web applications.

Key capabilities include:

* Simulating multiple users
* Measuring response times
* Monitoring server performance
* Generating performance reports

---

## Test Scenario

The test simulates multiple users sending HTTP requests to a web application.

Configuration:

* Number of Users: 500
* Ramp-Up Time: 1 seconds
* Loop Count: 1

The test evaluates how the system performs under concurrent load.

---

## Performance Metrics

The following metrics were analyzed:

* Response Time
* Throughput
* Latency
* Error Rate

These metrics help identify performance issues and system limitations.
