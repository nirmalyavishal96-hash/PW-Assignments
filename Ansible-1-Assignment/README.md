# Project Overview

This project demonstrates the installation and configuration of **Ansible**, a popular configuration management tool used in DevOps environments.

The assignment focuses on:

- Installing Ansible
- Configuring the Ansible inventory
- Writing a basic Ansible playbook
- Automating the installation and management of an Nginx web server

# Tools and Technologies Used

| Tool | Purpose |
|-----|------|
| Ansible | Configuration Management |
| Ubuntu Linux | Environment |
| Nginx | Web Server |
| YAML | Playbook configuration language |
