# Python Scripting Assignment

## Overview

This repository contains solutions for the **Python Scripting Assignment**.
The objective of this assignment is to demonstrate basic scripting techniques in Python including **file handling, data processing, and web scraping**.

The assignment is implemented using **Python scripts in a Jupyter / Google Colab notebook**.

## Tasks Implemented

### 1. Basic File Operations and Data Processing

A Python script was written to perform common file handling operations such as:

* Creating files
* Writing data to files
* Reading data from files
* Processing numerical data from a file

Example operations implemented:

* Writing text into a file
* Reading file contents
* Calculating the sum of numbers stored in a file

---

### 2. Simple Web Scraper

A basic web scraping script was developed using the following Python libraries:

* `requests`
* `BeautifulSoup`

The scraper performs the following tasks:

* Sends an HTTP request to a website
* Parses HTML content
* Extracts useful information such as:

  * Page title
  * Links from the webpage
